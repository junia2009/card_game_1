# 3D Solitaire – Klondike

宇宙空間に浮かぶテーブルで遊ぶ、Three.js 製の 3D クロンダイク・ソリティア。  
PWA 対応でスマートフォンのホーム画面にも追加でき、オフラインでもプレイできます。

🎮 **プレイ**: https://junia2009.github.io/card_game_1/

---

## 特徴

### ゲームプレイ
- **完全なクロンダイク・ルール** — 場札7列・組札4列・山札/捨て札の全操作に対応
- **スコアシステム** — 組札に積む・カードをめくるなど操作ごとにスコアが加算
- **手数カウント** — 全移動操作を記録
- **アンドゥ** — 最大30手まで巻き戻し可能（スコア・手数も復元）
- **ヒント機能** — 有効な手があるカードをハイライト表示
- **クリア演出** — 紙吹雪アニメーション＋スコア表示
- **新しいゲーム** — スコア・手数・履歴を完全リセット

### ビジュアル
- **Three.js 3D レンダリング** — カードが立体的に並ぶテーブル
- **宇宙空間背景**
  - 6,000 星 + 3,000 天の川（GLSL ShaderMaterial・瞬き表現）
  - 7 星雲スプライト（多層構造、呼吸する opacity アニメーション）
  - 3 遠方銀河スプライト
  - 最大 3 本同時の流れ星
  - 惑星（帯雲テクスチャ + 大気圏グロー + 専用 PointLight）
  - 小惑星ベルト 40 個（自転・公転付き）
  - 神秘的エネルギー体 × 4（光輪・クリスタル・プラズマ球・ダイヤモンド）が Lissajous 軌跡で組札奥を漂う
- **宇宙テーブル** — ホログラフィックグリッドテクスチャ＋半透明＋縁の呼吸する発光フレーム

### サウンド
Web Audio API によるプログラム合成（外部ファイル不要）

| 効果音 | タイミング |
|--------|-----------|
| クリック音 | カード選択・山札めくり |
| フリップ音 | カードを裏から表にめくる |
| 配置音 | 場札への移動成功 |
| 組札音 | 組札への積み上げ |
| エラー音 | 無効な移動 |
| クリア音 | 全枚クリア |

### PWA・モバイル対応
- ホーム画面への追加（Android / iOS）
- Service Worker によるオフラインキャッシュ
- `viewport-fit=cover` でノッチ・Dynamic Island 対応
- セーフエリア対応 CSS（`env(safe-area-inset-*)`)
- ポートレート・ランドスケープ両対応（`camera.up` による Three.js 内部回転）
- タッチ入力（`pointerup` + 10 px 閾値で誤タップを防止）

## 操作方法

| 操作 | 内容 |
|------|------|
| カードをクリック/タップ | 選択（選択済みで再クリックすると解除） |
| 移動先をクリック/タップ | カードを移動 |
| 山札クリック | 1枚めくって捨て札へ |
| **新しいゲーム** | ゲームをリセット |
| **ヒント** | 移動可能なカードをハイライト |
| **元に戻す** | 1手アンドゥ |
| **？ 遊び方** | ルール説明を表示 |

### ルール（クロンダイク）
1. 場札列は **赤黒交互・1つ小さい数字** のカードのみ積める
2. キング（K）は空の列に移動できる
3. エース（A）から同マーク順に **組札**（画面奥）に積んでいく
4. 全 52 枚を組札に積めばクリア

| 項目 | 内容 |

## ローカルで動かす

```bash
# 任意の HTTP サーバで index.html を配信するだけで動作します
# 例: Python
python -m http.server 8080

# 例: Node.js npx
npx serve .
```

> `file://` プロトコルでは Service Worker が動作しないため、ローカルサーバ経由を推奨します。
